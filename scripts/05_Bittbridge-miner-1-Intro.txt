So what is BittBridge? It's a decentralized-AI network built on top of the Bittensor infrastructure, and the subnet is basically the project. This project is for New England energy-demand prediction. Here's the high-level workflow of how everything works behind the scenes. There are two main actors: the validators and the miners. The validators send a challenge to the miner — they just ask, "Hey miner, can you make a prediction of what the energy demand is going to be in, say, the next five minutes?" The miner sees that the validator is asking for a prediction for this timestamp, uses a machine-learning model (or any prediction model) to make one, and sends it back to the validator. Then, when the validator gets the actual information — what the real energy demand was at that timestamp — it has both the actual data and the miner's predicted data. Using the incentive mechanism, the validator evaluates the miner's performance and sends weights to the Bittensor blockchain, which holds all the rewards for the miners.
